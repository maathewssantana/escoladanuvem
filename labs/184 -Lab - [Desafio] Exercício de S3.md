# Exercício de S3 🖥️

## Laborátório 🥼

## Objetivo

- Criar um bucket do S3. 

- Fazer upload de um objeto para esse bucket. 

- Acessar o objeto usando um navegador. 

- Listar o conteúdo do bucket do S3 utilizando a AWS Command Line Interface (AWS CLI).
